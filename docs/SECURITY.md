# How to Report a Vulnerability

If you believe you have found a vulnerability in Chhoto URL you can report it using one of the following ways:

- Contact me at [mail@sayantansantra.com](mailto:mail@sayantansantra.com).
- Create a [security advisory](https://github.com/SinTan1729/chhoto-url/security/advisories).

Please do not report security vulnerabilities through public GitHub issues.

# Supported Versions

Only the [latest version](https://github.com/SinTan1729/chhoto-url/releases/latest) is supported.
